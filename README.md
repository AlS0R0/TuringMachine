# TuringMachine

Симулятор машины Тьюринга с графическим интерфейсом на Qt (поддерживаются версии 5 и 6).

## Возможности

- Задание пользовательского алфавита (основные и дополнительные символы)
- Табличное описание правил переходов (состояния, читаемый символ → действие)
- Визуализация бесконечной ленты с анимацией перемещения каретки
- Пошаговое выполнение, автоматический режим с регулировкой скорости
- Сброс и перезапуск машины с исходной строкой

## Сборка

Требуется CMake ≥ 3.16, компилятор с поддержкой C++17, Qt (Widgets).
